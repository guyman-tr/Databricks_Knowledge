# Moneyfarm V2 - HLD

_pageId=12216961926 spaceId=752091711 url=https://etoro-jira.atlassian.net/wiki/spaces/XP/pages/12216961926/Moneyfarm+V2+-+HLD_

## High Level Design

The purpose of Moneyfarm project is to direct users to register on Moneyfarm platform.

The criteria for such users are:

1. countryID=UK
2. designatedRegulation=FCA
3. playerStatus is Normal
4. User has at least one Approved deposit
5. The user isn't a "Legacy user" (didn't register with Moneyfarm from eToro in the past)

The criteria for such users can be checked either on the client or BE side.

## Configuration (CCM)

| **CCM key** | **Value (prod)** | **Description** |
|---|---|---|
| isaAccount.moneyfarmV2.isEnabledGeneral | boolean | Feature flag - general |
| isaAccount.moneyfarmV2.shouldTemporaryRedirectLegacyUserFromDeepLink | boolean | Temporary redirect legacy users from deep links even if the general feature flag is off |
| isaAccount.moneyfarmV2.enabledGcidWhitelist | Array of numbers (e.g. [7956, 4576, 5329]) | A feature flag for the white list of GCIDs |
| isaAccount.moneyfarmV2.redirectUrl | https://etoro.moneyfarm.com/?authorization_code={{authorizationCode}} | Redirect URL for Eligible / Registered user |
| isaAccount.moneyfarmV2.redirectUrlForLegacyUser | https://app.moneyfarm.com/gb/sign-in | Redirect URL for Legacy user |
| isaAccount.moneyfarmV2.termsAndConditionsUrl |  | T&C checkbox - terms and condition URL |
| menu.shouldShowSideNavIsaAccountNewLabel | boolean | Whether to display the "New" label in the side menu (default=false) |
