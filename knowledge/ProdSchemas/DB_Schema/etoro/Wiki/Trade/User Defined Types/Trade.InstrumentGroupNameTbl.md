## 1. Business Meaning

Trade.InstrumentGroupNameTbl is a table-valued parameter for inserting NEW instrument group definitions. No GroupID is present because the ID is auto-generated on insert. GroupName and Description use binary collation (Latin1_General_BIN). Counterpart to InstrumentGroupNameAndIDTbl which is for updates and includes GroupID.

---

## 2. Business Logic

### 2.1 Insert new instrument groups

**What**: The TVP passes rows with GroupName and optional Description. InsertTradingInstrumentGroupName creates new rows in Trade.InstrumentGroups with auto-generated GroupIDs.

**Columns/Parameters Involved**: GroupName, Description

**Rules**: GroupName required; Description optional. Both use Latin1_General_BIN. Cannot specify GroupID - generated by target table.
