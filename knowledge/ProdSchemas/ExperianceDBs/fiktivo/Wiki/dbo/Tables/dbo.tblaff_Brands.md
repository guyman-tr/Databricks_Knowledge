## 1. Business Meaning

This table defines the platform brands under which marketing assets are organized. Each brand corresponds to a legal entity or product line (eToro global, eToro EU, eToro USA, eToro Partners). Banners and media groups are tagged with a BrandID to ensure affiliates receive creative assets appropriate for their regulated jurisdiction.

Without this table, the platform could not separate marketing materials by regulatory entity. An EU-regulated affiliate seeing US-specific banners would create compliance risk. Brands are managed by admin users with Brands_* permissions.

Rows are created/updated via dbo.UpdateInsertBrand. The table contains 4 real brand entries plus ~32 test data entries generated by automated testing (identifiable by random character BrandName values). The unique constraint on BrandName prevents duplicate brand definitions.

## 8. Sample Queries

### 8.1 List real brands (exclude test data)
```sql
SELECT BrandID, BrandName
FROM dbo.tblaff_Brands WITH (NOLOCK)
WHERE BrandName LIKE 'eToro%'
ORDER BY BrandID
```

### 8.2 Count banners per brand
```sql
SELECT b.BrandID, br.BrandName, COUNT(b.BannerID) as BannerCount
FROM dbo.tblaff_Banners b WITH (NOLOCK)
JOIN dbo.tblaff_Brands br WITH (NOLOCK) ON b.BrandID = br.BrandID
GROUP BY b.BrandID, br.BrandName
ORDER BY BannerCount DESC
```

### 8.3 Find media groups for a specific brand
```sql
SELECT g.GroupID, g.GroupName, br.BrandName
FROM dbo.tblaff_Groups g WITH (NOLOCK)
JOIN dbo.tblaff_Brands br WITH (NOLOCK) ON g.BrandID = br.BrandID
WHERE br.BrandName = 'eToro'
```
