## 1. Business Meaning

dbo.tblaff_SocialMessages captures social trading events generated by eToro's social investment network features. These include social feed posts, copy-trading interactions, competition results, and other social activity that may trigger affiliate tracking or attribution. Each message records the event type, involved customer, instrument, and various contextual parameters.

This table bridges eToro's social trading features with the affiliate system. Social activity by referred customers generates events that can be tracked for affiliate analytics and potentially influence commission calculations based on social engagement metrics.

Data is generated when social events occur on the platform. The table contains ~97K records with two message types: Type 2 (91%, ~88K records) and Type 1 (9%, ~8.7K records). The 10 optional fields (Optional1-10) provide flexible extension for different message type payloads.

## 2. Business Logic

### 2.1 Social Message Type Classification

**What**: Two types of social events tracked for affiliate purposes.

**Columns/Parameters Involved**: `MessageTypeID`

**Rules**:
- MessageTypeID=2: Dominant type (91%, 88K records) - likely social feed activity or copy-trading events
- MessageTypeID=1: Secondary type (9%, 8.7K records) - likely a different social interaction category
- No explicit lookup table found for MessageTypeID

### 2.2 Social Trading Context

**What**: Each message captures rich context about the social trading event.

**Columns/Parameters Involved**: `Network`, `GCID`, `CID`, `PID`, `PositionID`, `IsBuy`, `InstrumentID`, `FromCID`, `Gain`, `Rank`

**Rules**:
- `Network`: Social network identifier (eToro has internal social networks/feeds)
- `GCID`/`CID`: The customer involved in the social event
- `PositionID`/`IsBuy`/`InstrumentID`: Trading position context (if the social event relates to a trade)
- `FromCID`/`FromUserName`: The originator of the social message (for copy-trading or shared posts)
- `Gain`/`TotalPrizes`/`Prize`: Competition/performance metrics
- `Rank`: Customer ranking in competitions or leaderboards

       UserGeneratedContent, Gain, IsReal
