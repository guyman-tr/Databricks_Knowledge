## 1. Business Meaning

dbo.tblaff_eCost_Commissions records affiliate commissions from eCost events (tblaff_eCost). eCost represents the effective cost/revenue share model where affiliates earn a percentage of the platform's revenue generated by their referred customers. This differs from fixed CPA commissions - eCost is variable and based on actual customer trading activity.

Contains ~1.47M records, making it the third-largest commission table. Triggers enforce RI: eCostID must exist in tblaff_eCost and AffiliateID must exist in tblaff_Affiliates. Includes an `eCost` field for the platform's effective cost value alongside the affiliate commission.

## 2. Business Logic

### 2.1 Multi-Tier eCost Commission

**Columns/Parameters Involved**: `eCostID`, `AffiliateID`, `Commission`, `Tier`, `eCost`

**Rules**:
- eCostID references tblaff_eCost. Triggers enforce RI on INSERT/UPDATE.
- Standard multi-tier distribution: Tier 1-5
- `eCost` field tracks the platform's effective cost value for this event (may differ from Commission)
- Revenue share model: Commission = eCost * affiliate's agreed percentage

### 2.2 Payment Lifecycle

- Same pattern: Paid=0/PaymentID=0 = unpaid, Paid=1/PaymentID>0 = paid

## 8. Sample Queries

### 8.1 Unpaid eCost commissions by affiliate
```sql
SELECT AffiliateID, Tier, COUNT(*) AS Records, SUM(Commission) AS TotalUnpaid
FROM dbo.tblaff_eCost_Commissions WITH (NOLOCK)
WHERE Paid = 0
GROUP BY AffiliateID, Tier ORDER BY TotalUnpaid DESC
```

### 8.2 eCost vs Commission analysis (revenue share ratio)
```sql
SELECT AffiliateID,
       SUM(eCost) AS TotaleCost,
       SUM(Commission) AS TotalCommission,
       CASE WHEN SUM(eCost) > 0 THEN SUM(Commission) / SUM(eCost) * 100 ELSE NULL END AS CommissionPct
FROM dbo.tblaff_eCost_Commissions WITH (NOLOCK)
WHERE eCost IS NOT NULL AND eCost > 0 AND Tier = 1
GROUP BY AffiliateID ORDER BY TotaleCost DESC
```

### 8.3 eCost commissions with event details
```sql
SELECT ec.Commission, ec.Tier, e.AffiliateID, e.Occurred
FROM dbo.tblaff_eCost_Commissions ec WITH (NOLOCK)
JOIN dbo.tblaff_eCost e WITH (NOLOCK) ON ec.eCostID = e.eCostID
WHERE ec.AffiliateID = @AffiliateID AND ec.Paid = 0
```
